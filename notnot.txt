sonra cd GitClone yazdı hoca ama not alırken kaçırdım ne işe yarıyor

Sonra git push yazıp değişikliği repomuza gönderiyoruz
bir daha pushlamak için git add. ve git commit -m"v?" yazmadan
tek seferde git commit -am"v?" yazıp ardından pushlayabiliriz
